Interactive button component with multiple style variants.
Buttons support icons, loading states, and full-width display.

## Guidelines

- Use **primary** for the main call-to-action on a page
- Use **outline** for secondary actions
- Use **danger** sparingly — only for destructive actions like delete

:::alert{type="info" title="Accessibility"}
Always provide visible text labels. Icon-only buttons require an `aria-label` attribute.
:::

## Keyboard Interaction

| Key | Action |
|-----|--------|
| `Enter` | Activates the button |
| `Space` | Activates the button |
